Overview

This project investigates the extent of side-channel analysis attacks on real-time reconstructing displays. It focuses mainly on recovering the screen contents of the external LCD display to which the PC is connected to. MatGrabber, a MATLAB application that reconstructs LCD screens in real time using techniques such as auto-correlation and synchronisation detection, was created based on existing approaches from prominent academic works.
